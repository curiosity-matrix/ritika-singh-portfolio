## 🧠 What Makes This Portfolio Different? (Holographic HUD)

Instead of a plain, standard page that "normal AI/ML students" build, this is a **Holographic Spatial Interface** designed to look like spatial computing:

- **Global 3D Screen Tilt:** The entire webpage is suspended in a virtual 3D space (`perspective: 1500px`). As you move your mouse, the **whole page tilts and rotates** to track your cursor, looking like a hovering glass tablet.
- **3D Hyperspace Data Tunnel:** The background is an active HTML5 Canvas simulation. Particles (data nodes) are plotted in 3D coordinate space ($X, Y, Z$) and **fly directly forward at the camera**. They grow, glow, and connect into moving neural rings as they get closer.
- **Mouse Spotlight Nebula:** Move your cursor around—a soft cosmic violet/indigo space nebula drifts in parallax behind the floating console, illuminating different sectors of the neural starfield like a spotlight.
- **`preserve-3d` Layer Pop-outs:** Cards and stats aren't flat. When you hover over my GPA, skills, or projects, the text, large values, and icons **physically lift off the card face** (`translateZ(40px)`). The elements literally float in front of the card background!

## ⚙️ Core Technical Pipeline
```text
Raw HTML/CSS → 3D Perspective Matrix Mappings → Canvas Projection Loops → Dynamic Stacking Fixes → Translucent Holo Viewport
```

*Why no external UI frameworks?* I wanted complete control over the performance and layout. Using heavy React/Next.js setups for a single-page portfolio would cause "spin-up delays" and page lag. I built this in **Vanilla JS and pure CSS variables** so it loads instantly (in milliseconds!) for recruiters.

## 📂 The Missions (Featured Projects)

Every project card inside my portfolio connects directly to its respective codebase on my GitHub profile:

1. **[Employee-Attrition-Predictor](https://github.com/curiosity-matrix/Employee-Attrition-Predictor) 🧠**
   * *The Problem:* Predicting which employees are about to leave before they do.
   * *The Code:* A 4-layer ANN model in TensorFlow/Keras with SMOTE class balancing. 83% overall accuracy with decision threshold tuned from 0.5 to 0.3 to maximize recall of at-risk employees.
2. **[behavioral-personality-classifier](https://github.com/curiosity-matrix/behavioral-personality-classifier) 🎭**
   * *The Problem:* Can a machine understand your character from behavioral traits?
   * *The Code:* Compares 4 ML models (SVM, KNN, Random Forest, Logistic Regression) on 20,000 rows. KNN won at 99.77% accuracy. Implements skewness-based null imputation (mean for normal features, median for skewed features) because blindly filling everything with mean is wrong.
3. **[SQL-_project1_retail_sales](https://github.com/curiosity-matrix/SQL-_project1_retail_sales) 🛒**
   * *The Problem:* Uncovering sales patterns from 2,000+ transactional records.

4. **[Online-Retails-Virtual-Experience-by-Tata-](https://github.com/curiosity-matrix/Online-Retails-Virtual-Experience-by-Tata-) 📊**
   * *The Problem:* Visualizing retail trends for simulated C-suite executives.
   * *The Code:* A 4-chart interactive Power BI dashboard highlighting seasonal revenue fluctuations, invalid quantity cleanups, and macro visual insights.
5. **[inventory-management-rule-based-](https://github.com/curiosity-matrix/inventory-management-rule-based-) ⚙️**
